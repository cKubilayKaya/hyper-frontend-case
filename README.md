# HYPER TEKNOLOJİ – FRONTEND CASE

### ENV

Projeye bir `.env` dosyası oluşturup aşağıdaki alanları eklemeniz gerekir.

```bash
NEXT_PUBLIC_API_URL=string
NEXT_PUBLIC_TOKEN=string
```

### Projeyi Başlat

Projenin bağımlılıkları indirin.

```bash
npm install
```

Projeyi başlatın.

```bash
npm run dev
```
