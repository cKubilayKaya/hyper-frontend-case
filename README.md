# HYPER TEKNOLOJİ – FRONTEND CASE

### ENV

Projenin kök dizininde bir `.env` dosyası oluşturup aşağıdaki alanları eklemeniz gerekir.

```bash
NEXT_PUBLIC_API_URL=string
NEXT_PUBLIC_TOKEN=string
```

### Projeyi Başlat

Projenin bağımlılıklarını indirin.

```bash
npm install
```

Projeyi başlatın.

```bash
npm run dev
```
